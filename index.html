String Methods

1. String.charAt()

Retruns a String Representing the character at a given index.

2. String.endsWith()

Retruns true if the string ends with the given string otherwise false.

3. String.includes()

Retruns true if the string contain the given string otherwise false.

4. String.indexOf()

Retruns the index within the string of first occurrance of the specified value or -1 if not found.

5. String.padEnd()

Retruns a new string with some content padded to the end of the string.

6. String.padStart()

Retruns a new string with some content padded to the start of the string.

7. String.repeat()

Returns a new string which contain the specified number of copies of the string.

8. String.replaceAll()

returns a new string with all values replaced.

9. String.slice()

Return a new string containing the character of string from the given index to the end of string .

10. String.split()

Returns an array of string spilt at the given index.

11. String.tostring()

Return the string representation of the specified object.

12. String.trim()

Return a new string with the leading and tralling whitespace removed.
